работа NUCLEO STM32F303 с микросхемой eeprom памяти 24LC64 по протоколу I2C.
память смонтирована на внешней отладочной плате.
нуклео записывает на микруху массив данных, потом считывает их из памяти с того же адреса. отправленный и считанный массивы сравниваются между собой. операции сопровождаются индикацией светодиода.
